# 변경 이력 — 2026-04-10

## [초기 릴리스] 수면 설문지 시스템 v1.0

---

### 새로운 기능

- **메인 페이지 (`index.html`)**
  - 환자번호, 생년월일, 두통 여부 입력 폼
  - 만 나이 자동 계산 및 표시
  - 나이·두통 여부에 따른 설문지 자동 배정
    - 만 6세 이상: SDQ → K-CSHQ → (PED-MIDAS) → PDSS
    - 만 6세 미만: K-CSHQ → BISQ-R → (PED-MIDAS)
  - 이전 세션 복원 배너 (이어서 진행 / 새 환자로 시작)

- **설문지 페이지 공통**
  - 상단 고정 내비게이션 바: 메인 이동, 환자 정보 표시, 설문 탭 (완료 시 ✓ 표시), 결과 보기, 결과 출력(Excel)
  - 이전/다음 설문 이동 버튼 (첫 설문 — 이전 없음, 마지막 설문 — 다음 없음)
  - 점수 계산 버튼 (컨트롤 패널 하단)
  - 재방문 시 이전 입력 답변 및 점수 자동 복원
  - 초기화 버튼, 홈으로 버튼

- **다중 환자 지원**
  - 마지막 설문 완료 후 **+ 새 환자 추가** 버튼
  - 환자별 결과 누적 저장 (브라우저 세션 내)

- **결과 보기 모달**
  - 상단 바의 **결과 보기** 버튼으로 현재까지 입력된 모든 환자 점수를 표로 미리 확인

- **Excel 통합 출력**
  - 전체 환자 결과를 단일 xlsx 파일로 출력
  - 시트 구성: 결과요약, SDQ, K-CSHQ, PED-MIDAS, BISQ-R, PDSS

- **포함 설문지**
  - SDQ (강점·난점 설문지, 25문항, 역채점 항목 포함)
  - K-CSHQ (소아 수면 습관 설문지, 46문항, 주관식 포함)
  - BISQ-R (영아 수면 설문지, 34문항)
  - PED-MIDAS (두통 영향 설문지, 6문항)
  - PDSS (주간 졸림 척도, 8문항, 역채점 항목 포함)

---

### 기술 사항

- `file://` 프로토콜 환경에서 localStorage 격리 문제를 URL 파라미터(sn_s, sn_r, sn_a) base64 인코딩 전달로 해결
- 공통 모듈 `js/survey-nav.js` 분리: 내비게이션, 세션 관리, 결과 저장, Excel 출력 담당
- xlsx.js (v0.18.5) CDN 사용
