# 수면 설문지 시스템 (SleepSurvey)

소아 수면 클리닉용 웹 기반 설문지 시스템입니다.  
환자 정보(나이, 두통 여부)를 입력하면 필요한 설문지 조합이 자동으로 결정되며, 결과를 Excel 파일로 출력할 수 있습니다.

---

## 포함 설문지

| 약어 | 설문지명 | 문항 수 | 대상 |
|------|----------|---------|------|
| SDQ | 강점·난점 설문지 | 25문항 | 만 6세 이상 |
| K-CSHQ | 소아 수면 습관 설문지 | 46문항 | 전 연령 |
| PED-MIDAS | 두통 영향 설문지 | 6문항 | 두통 환자 |
| BISQ-R | 영아 수면 설문지 | 34문항 | 만 6세 미만 |
| PDSS | 아동용 주간 졸림 척도 | 8문항 | 만 6세 이상 |

---

## 설문 배정 기준

환자의 **만 나이**와 **두통 여부**에 따라 설문 조합이 자동 결정됩니다.

| 조건 | 배정 설문지 |
|------|------------|
| 만 6세 이상 · 두통 없음 | PDSS → SDQ → K-CSHQ |
| 만 6세 이상 · 두통 있음 | PED-MIDAS → PDSS → SDQ → K-CSHQ |
| 만 6세 미만 · 두통 없음 | K-CSHQ → BISQ-R |
| 만 6세 미만 · 두통 있음 | PED-MIDAS → K-CSHQ → BISQ-R |

## 실행 방법

별도 서버 설치 없이 브라우저에서 바로 실행할 수 있습니다.

1. 저장소를 다운로드하거나 클론합니다.
   ```
   git clone https://github.com/yune1ha/SleepSurvey.git
   ```
2. `index.html` 파일을 브라우저(Chrome 권장)로 엽니다.

> **주의:** `file://` 프로토콜에서도 정상 동작하도록 설계되어 있습니다.

---

## 사용 방법

### 1단계 — 환자 정보 입력

메인 화면에서 다음 정보를 입력합니다.

- **병원 환자번호** (숫자)
- **생년월일** (만 나이 자동 계산)
- **두통 여부** (예 / 아니오)

입력 완료 후 **확인** 버튼을 누릅니다.

### 2단계 — 설문 진행

- 필요한 설문지 목록이 자동으로 표시됩니다.
- **시작** 버튼으로 첫 번째 설문을 시작합니다.
- 각 설문 완료 후 **점수 계산/저장** 버튼을 누르면 결과가 저장됩니다.
- 상단 내비게이션 바의 설문 탭을 클릭하거나 **이전/다음 설문** 버튼으로 이동합니다.
- 저장된 설문은 탭에 **✓** 표시가 나타납니다.

### 3단계 — 결과 출력

- 상단 내비게이션 바의 **결과 보기** 버튼으로 현재까지의 결과를 모달 창에서 확인합니다.
- **결과 출력 (Excel)** 버튼을 누르면 `SleepSurvey_YYYY-MM-DD.xlsx` 파일이 다운로드됩니다.

#### Excel 파일 구성

| 시트 | 내용 |
|------|------|
| 결과요약 | 환자별 각 설문 총점 요약 (1행 1환자) |
| SDQ | SDQ 문항별 응답 및 총점 |
| K-CSHQ | K-CSHQ 영역별 점수 및 주관식 답변 |
| PED-MIDAS | PED-MIDAS 문항별 응답 및 총점 |
| PDSS | PDSS 문항별 응답 및 총점 |
| BISQ-R | BISQ-R 항목별 응답 |

### 다중 환자 입력

- 마지막 설문 화면에서 **+ 새 환자 추가** 버튼을 누르면 현재 환자 데이터가 누적 저장됩니다.
- 메인 화면으로 돌아가 다음 환자를 입력합니다.
- Excel 출력 시 누적된 모든 환자의 데이터가 한 파일에 포함됩니다.

## 세션 복원

브라우저를 실수로 닫거나 새로고침한 경우, 다음 접속 시 **이전 세션 복원 배너**가 표시됩니다.

- **이어서 진행**: 중단 지점부터 설문을 재개합니다.
- **새 환자로 시작**: 이전 세션을 삭제하고 새로 시작합니다.
