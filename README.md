# 수면 설문지 시스템

소아 수면 클리닉을 위한 웹 기반 설문지 통합 관리 시스템입니다.  
환자의 나이와 두통 여부에 따라 필요한 설문지를 자동으로 배정하고, 결과를 Excel 파일로 통합 출력합니다.

---

## 포함 설문지

| 설문지 | 대상 | 문항 수 | 비고 |
|--------|------|---------|------|
| SDQ (강점·난점 설문지) | 만 6세 이상 | 25문항 | 일부 역채점 |
| K-CSHQ (소아 수면 습관 설문지) | 전 연령 | 46문항 | 주관식 포함 |
| BISQ-R (영아 수면 설문지) | 만 6세 미만 | 34문항 | 영아 전용 |
| PED-MIDAS (두통 영향 설문지) | 두통 있는 환자 | 6문항 | 선택적 배정 |
| PDSS (주간 졸림 척도) | 만 6세 이상 | 8문항 | 일부 역채점 |

### 설문 배정 기준

- **만 6세 이상**: SDQ → K-CSHQ → (PED-MIDAS, 두통 시) → PDSS
- **만 6세 미만**: K-CSHQ → BISQ-R → (PED-MIDAS, 두통 시)

---

## 실행 방법

별도의 서버 설치 없이 브라우저에서 바로 실행할 수 있습니다.

1. `index.html` 파일을 브라우저에서 열기 (더블클릭 또는 드래그 앤 드롭)
2. 환자 정보 입력 (환자번호, 생년월일, 두통 여부)
3. **확인** 버튼을 누르면 해당 설문지 목록이 자동으로 표시됨
4. **시작** 버튼으로 첫 번째 설문 진행
5. 각 설문 완료 후 **점수 계산** → **다음 설문** 으로 순서대로 이동
6. 마지막 설문까지 완료하면 **결과 출력 (Excel)** 버튼으로 통합 파일 저장

---

## 다중 환자 입력

마지막 설문 페이지 하단의 **+ 새 환자 추가** 버튼을 누르면 현재 환자 결과가 저장되고 홈 화면으로 돌아갑니다.  
이후 새 환자 정보를 입력하여 반복 진행할 수 있으며, **결과 출력 (Excel)** 버튼을 누를 때 모든 환자의 결과가 한 파일에 합산 출력됩니다.

---

## 결과 확인 (미리 보기)

상단 내비게이션 바의 **결과 보기** 버튼을 누르면 현재까지 입력된 모든 환자의 점수를 표 형태로 미리 볼 수 있습니다.

## 파일 구조

```
SleepSurvey/
├── index.html          # 메인 진입점 (브라우저에서 열기)

├── surveys/
│   ├── SDQ.html
│   ├── K-CSHQ.html
│   ├── BISQ-R.html
│   ├── PED_MIDAS.html
│   └── PDSS.html

```

## 주의 사항

- 데이터는 브라우저의 로컬 스토리지 및 URL 파라미터를 통해 페이지 간 전달됩니다.
- **브라우저를 닫거나 탭을 새로 열면 세션이 초기화**될 수 있으므로, 모든 설문 완료 후 반드시 Excel 출력을 먼저 진행하세요.
- 권장 브라우저: Chrome, Edge (최신 버전)
